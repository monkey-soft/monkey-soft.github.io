使用 Python 将数据写到 CSV 文件

  2018-05-26   2018-05-26
  Python 必知必会
   6 分钟 读完 (大约 927 个字)

我们从网上爬取数据，最后一步会考虑如何存储数据。如果数据量不大，往往不会选择存储到数据库，而是选择存储到文件中，例如文本文件、CSV 文件、xls 文件等。因为文件具备携带方便、查阅直观。
